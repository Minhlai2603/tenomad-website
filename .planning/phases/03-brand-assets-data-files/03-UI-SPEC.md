# Phase 3 UI Specification: Brand Assets

## Logo & Favicon
- **Double-N Logo:**
  - ViewBox: `0 0 140 36`
  - Colors: Orange-to-Yellow gradient (`#FFB800` to `#FF5C00`).
  - Font: `Inter`, weights `800`, uppercase.
- **Favicon:**
  - Standard favicon: 32x32px icon.
- **OG Image:**
  - Dimensions: `1200x630` px.
  - Background: Dark Indigo (`#1B2A4A` / `#0B0F19`).
  - Text/Glow: Gradient text and neon blue/purple glows with logo icon centered.

## Service Icons
- Lucide/Tabler style icons implemented as inline SVGs for high resolution:
  - Web Development: Code icon.
  - Mobile Development: Smartphone icon.
  - Dedicated Team: Users icon.
  - Startups: Rocket icon.
  - QA Testing: ShieldCheck/AlertCircle icon.
  - Design: Palette icon.
  - AI & NLP: Cpu/Activity icon.

## Team Avatar Styles
- Management members: webp photo or styled letter circles (double-circle border with gold gradient text initials).
- Team members: styled letter circles (white text on `--accent-blue` or `--accent-indigo` background, centered text).
- Touch target: N/A (avatars are non-interactive images/elements).
