## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline padding (`--space-1`) |
| sm | 8px | Compact element spacing (`--space-2`) |
| md | 16px | Default element spacing (`--space-4`) |
| lg | 24px | Section card padding (`--space-6`) |
| xl | 32px | Layout gaps (`--space-8`) |
| 2xl | 48px | Major section breaks (`--space-12`) |
| 3xl | 64px | Page-level spacing (`--space-16`) |

Exceptions: `--space-3` (12px), `--space-5` (20px), `--space-10` (40px), `--space-20` (80px), `--space-30` (120px)

## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | #FFFFFF | Background, main body surfaces (`--bg-primary`) |
| Secondary (30%) | #F5F7FA | Cards, panels, footer/subpages backgrounds (`--bg-secondary`) |
| Accent (10%) | #0B88EE | Team member link hover states, core value underlines (`--accent-blue`) |
| Hero Accent | #E8A600 | Call to action highlights, header active links (`--accent-gold`) |
| Destructive | #EF4444 | Validation error boundaries |

Accent reserved for: Name hover shifts, active navigation markers, and CTA hover highlights.
