## Extracted Decisions

1. Header: Sticky + glassmorphism, logo left, nav center (hover underline), CTA "Let's Talk" right.
2. Homepage: 9 sections in defined order with specific animation types per section.
3. Footer: 3-column layout (company info, quick links, contact), bottom bar with copyright + social icons.
4. Service detail: Breadcrumb + split layout (sidebar nav left, content right) + related projects + CTA.
5. Portfolio: Filter tabs (All, Design, Website, Mobile, Startup, E-commerce, Hybrid App) + 3-col grid.
6. Careers: Accordion card pattern for job listings.
7. Blog: Split layout (2-col grid left, sidebar right) with search, categories, recent posts.
