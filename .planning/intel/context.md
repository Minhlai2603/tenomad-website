# Synthesized Context

## Project Overview

Tenomad is a software development agency based in Hue City, Vietnam. The project is a complete visual redesign and technical rebuild of their existing WordPress website into a modern, high-performance static site. The goal is to create a premium agency showcase that impresses prospective clients and attracts talent.

## Business Goals

1. Impress prospective clients with modern, premium web design
2. Showcase the agency's portfolio of 11+ projects across web, mobile, and hybrid platforms
3. Attract talent via a dedicated careers section with current job openings
4. Establish thought leadership through a blog with technical articles
5. Provide an accessible contact channel for project inquiries

## Target Audience

- Prospective clients evaluating software development partners
- Job seekers in Vietnam's tech industry (developers, designers)
- Industry peers evaluating Tenomad's capabilities

## Content Scope

- 15 pages total (Home, About, Services×8, Portfolio, Careers, Blog, Contact, 404)
- 4 blog posts (Content Collections)
- 11 portfolio projects
- 3 job listings (Vietnamese content)
- 8 management + 32 team members
- 3 client testimonials

## Technical Context

- Migrating from WordPress (legacy) to Astro v5 (SSG)
- All content extracted and documented in CONTENT.md
- Design tokens fully specified in DESIGN-SYSTEM.md
- Wireframes documented in UI-WIREFRAME.md
- Asset strategy documented in ASSETS.md (agent-generated, no stock photos)
- AGENTS.md serves as the single source of truth, overriding all other docs on conflicts
