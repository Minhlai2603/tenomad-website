# Synthesized Context

## Project Background

Tenomad is a software development agency based in Hue City, Vietnam. They currently have a WordPress website that needs a visual refresh and technical modernization. The rebuild will create a static marketing site showcasing their services (web, mobile, dedicated teams, startups, QA, design, AI/NLP), portfolio of 11 projects, team of 40+ people (8 management + 32 staff), and 3 open career positions.

## Business Goals

1. Impress prospective clients with a visually stunning, premium web presence
2. Showcase technical capabilities and portfolio to generate leads
3. Attract talent through modern careers section
4. Establish thought leadership via blog content
5. Maximize performance and SEO for organic discovery

## Technical Context

- Migrating FROM: WordPress (legacy, dynamic)
- Migrating TO: Astro v5 static site (SSG)
- Content source: Extracted WordPress REST API data (preserved in CONTENT.md)
- Domain: tenomad.com
- Hosting: Vercel (static)
- No CMS — content baked into data files and markdown

## Design Context

- Design direction: Modern, premium, "wow at first glance"
- Mode: Light overall with dark accent sections (hero, footer, CTA)
- Visual language: Glassmorphism, neon accents on dark sections, smooth GSAP animations
- Typography: Inter font family, Major Third scale (1.25 ratio)
- Color: Gold CTAs, blue brand accent, indigo dark sections, neon cyan/blue/purple glow effects

## Content Context

- 7 service offerings with full detail copy
- 11 portfolio projects with tags and descriptions
- 4 blog articles (digital nomads, iOS skills matrix, blockchain tutorial, entrepreneurship lessons)
- 3 job listings with Vietnamese descriptions
- 3 client testimonials
- 8 management profiles + 32 team member listings

## Stakeholder Context

- Primary audience: Prospective clients (US-based, English-speaking)
- Secondary audience: Job seekers (Vietnamese market)
- Site admin: Tenomad team (no CMS, code-level updates)
