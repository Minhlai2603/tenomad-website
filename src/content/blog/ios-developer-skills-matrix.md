This skills matrix outlines the technical and professional expectations across Junior, Middle, and Senior levels for iOS developers at Tenomad:

| Attribute | Junior Developer | Middle Developer | Senior Developer |
|---|---|---|---|
| **Core Ability** | Can write simple features with assistance. | Able to implement full modules independently. | Can architect complete systems from scratch. |
| **Code Integration** | Needs code review support for branch management. | Handles pull requests and resolves conflicts smoothly. | Manages release branches and CI/CD pipelines. |
| **Paradigms** | Understands OOP basics. | Proficient with POP (Protocol-Oriented) and Functional. | Expert in reactive programming and custom paradigms. |
| **Dependencies** | Uses CocoaPods basic configs. | Manages complex Podfiles and Swift Package Manager. | Builds custom libraries and frameworks. |
| **Platform Cocoa Touch** | Basic UIKit knowledge. | Advanced UIKit, custom transitions, Core Graphics. | Performance profiling, system memory optimization. |
| **Client-Server** | Knows how to use URLSession. | Proficient with Alamofire, JSON serialization, REST APIs. | High-performance sync, offline storage handling. |
| **Memory Management** | Basic understanding of ARC. | Avoids retain cycles, uses weak/unowned properly. | Profiles leaks, runs memory allocations diagnostics. |
| **User Interface** | Interface Builder & Storyboards. | Autolayout via code (SnapKit), custom UI elements. | Complex layout rendering, GPU optimizations. |
| **Multithreading** | Understands GCD (main queue vs global queue). | Operations & OperationQueues, race condition handling. | Synchronization locks, advanced concurrency patterns. |
| **Attitude** | Eager to learn, takes guidance. | Proactive, team player, mentors juniors. | Leader, takes project ownership, solves hard problems. |
| **Design Patterns** | Knows MVC. | Proficient with MVVM, VIPER, Delegate, Singleton. | Designs custom architectural templates for projects. |
| **Product Quality** | Writes code that runs. | Adds unit tests, focuses on code readability. | Directs QA strategy, ensures robust crash-free metrics. |
